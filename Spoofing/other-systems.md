# Spoofing — Other systems (suggestions)

- Mobile apps: device-bound tokens, jailbreak/root detection signals, refresh token rotation tests.
- Message queues: publisher/consumer identity, topic permissions, dead-letter routing permissions.
- Kubernetes: service account token scope, RBAC impersonation tests, namespace isolation.
- LLM/AI systems: prompt “identity” boundaries (who can instruct the model), tool-use authorization checks.
